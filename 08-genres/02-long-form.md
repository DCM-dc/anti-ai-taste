# 08-genres / 02 · Long-form

## Scope

Essays, journalism, blog posts > 1500 words, opinion pieces, reported
features, whitepapers.

This genre needs **sustained paragraphs and developed arguments**.
Several universal rhythm rules relax here.

---

## Universal rules that apply directly

- **Language:** signature-word density ≤ 3 per 500 words.
- **Language:** no "not X but Y" reflex (≤ 1 per piece).
- **Rhetoric:** no decorative quotes; quote only when load-bearing.
- **Rhetoric:** no forced elevation at the end.
- **Rhetoric:** no coined terms without lexical gap.

---

## Genre-specific overrides

### Override 1: RELAX the fragment rule

The universal rhythm rule (`03-language/03-rhythm.md`) says "use fragments
per 500 words." For long-form, **reduce fragment density**.

Long-form readers are in for the long haul. They want sustained
paragraphs that develop ideas. Too many fragments makes the prose feel
choppy and show-off-y.

| Genre | Fragments per 500 words |
|---|---|
| Universal default | 1–2 |
| Long-form | 0–1 (use sparingly for emphasis) |

Fragments are still allowed for **punch** ("He paused." or "None of it
worked.") — just not as a regular rhythm.

### Override 2: RELAX the "no transition words" rule

The universal rule says transition words (furthermore, moreover,
additionally) should be ≤ 2 per 500 words. For long-form, **this can
relax slightly** — but the bar is still low.

Long-form sometimes needs explicit transitions to bridge sections,
especially in reported features that jump between scenes. The rule:

- Replace "furthermore / moreover" with **specific transitions**:
  "Three days later," "In the next room," "The opposite happened."
- Time markers and place markers are better than abstract connectors.
- If you must use an abstract connector, use it once, not as a reflex.

| Genre | Abstract transitions per 500 words |
|---|---|
| Universal default | ≤ 2 |
| Long-form | ≤ 2 (same — but allow more time/place markers) |

### Override 3: ALLOW longer paragraphs

The universal rhythm rule warns against uniform 3–5 sentence paragraphs.
For long-form, **paragraphs can be longer** — 5–8 sentences is normal
for essayistic prose.

The rule is still **variance**: include some 1–2 sentence paragraphs
for emphasis, some 6–8 sentence paragraphs for development. Don't make
every paragraph 5 sentences.

### Override 4: REQUIRE sustained argument

Unlike marketing (punchy, fragmented) or technical docs (procedural,
complete), long-form needs **developed argument**. Each paragraph should:

- Make a claim
- Provide evidence (specific: numbers, names, dates, quotes)
- Analyze the evidence (don't just present it)
- Connect to the next paragraph

The universal rule "every claim must be grounded" applies doubly here.
Long-form without specifics becomes an opinion dump.

## Genre-specific smells (in addition to universal tells)

### Smell 1: The "thread" structure that goes nowhere

```
✗ "I've been thinking about X. Here are 7 thoughts.
   1. ...
   2. ...
   ...
   7. ...
   What do you think?"
```

This is the AI-generated Twitter thread format. It's a list of
undeveloped opinions. Long-form requires **development**, not listing.

### Smell 2: The "I learned X, Y, Z" recap ending

```
✗ "In conclusion, I learned that perseverance matters, that friendship
  is important, and that the journey is the destination."
```

Forced elevation + list of platitudes. End on a specific moment or a
specific unresolved question.

### Smell 3: The "research shows" without research

Long-form often cites studies. The universal rule applies: name the
study, name the authors, name the year. "Research shows" is the AI tell
for "I'm making this up."

### Smell 4: The "let that sink in" interjection

```
✗ "Let that sink in."
✗ "Read that again."
✗ "Stop and think about that for a moment."
```

These are AI reflexes for emphasis. If the point needs emphasis, make
the point sharper — don't tell the reader how to react.

## Instruction template (long-form-specific)

```
Apply universal rules, plus:

FRAGMENTS:
  - Reduce to 0–1 per 500 words (universal default is 1–2)
  - Use only for genuine punch, not as regular rhythm

TRANSITIONS:
  - Abstract connectors (furthermore, moreover): ≤ 2 per 500 words
  - Time/place markers ("Three days later", "In the next room"): encouraged
  - Replace abstract with specific wherever possible

PARAGRAPHS:
  - Allow 5–8 sentence paragraphs for development
  - Still require variance: mix with 1–2 sentence paragraphs for emphasis
  - Don't make every paragraph the same length

ARGUMENT:
  - Each paragraph: claim → evidence (specific) → analysis → connection
  - No undeveloped lists (the "7 thoughts" thread)
  - No opinion dumps — every opinion needs grounding

ENDING:
  - No "I learned X, Y, Z" recap
  - No "let that sink in" interjections
  - End on a specific moment or specific unresolved question

CITATIONS:
  - Every study named (authors, year, venue)
  - Every statistic sourced
  - "Research shows" → name the research
```
